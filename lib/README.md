# Testing pipe error sample 

1. Create a file named `pipe_error_sample.sh`
2. Add executable permissions to the script
```bash
chmod +x lib/pipe_error_sample.sh 
```

3. Execute the script.
- lib/pipe_error_sample.sh
- bash -e lib/pipe_error_sample.sh
- bash --noprofile --norc -eo pipefail lib/pipe_error_sample.sh
